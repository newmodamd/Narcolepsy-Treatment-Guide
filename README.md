Narcolepsy is categorized into:
- **Type 1 Narcolepsy (NT1):** With cataplexy (sudden muscle weakness)
- **Type 2 Narcolepsy (NT2):** Without cataplexy
- **Idiopathic Hypersomnia:** A related sleep disorder
